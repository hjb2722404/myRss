    Usenet 兴衰史
    早在 Facebook 存在前，甚至早在互联网存在前，Usenet 就已经存在了。它是第一个社交网络。1979 年，杜克大学的 CS 学生 Tom Truscott 和 Jim Ellis 设想了一个在不同主题下共享消息的网络。这些消息（或称为文章/帖子）被递交到后来被称为新闻组的主题类别中。1980 年，Truscott 和 Ellis 使用 Unix to Unix Copy Protocol(UUCP)协议与北卡罗来纳大学建立了第一个 Usenet 节点。此后它在早期网络中迅速传播。消息储存在新闻服务器上，通过新闻服务器检索，服务器之间是对等的，会交换传输消息，在几小时内你的消息会传遍整个网络。Usenet 开发了自己的网络协议 NNTP 加快消息传输。今天基于 ActivityPub 去中心化联邦协议的社交网络就是采用类似方法，Meta 的 Threads 正在探索使用 ActivityPub 与 Mastodon 等联邦宇宙平台进行互联。历史在往复。 在 Google Groups 停止支持 Usenet 后，可免费使用的常用客户端是 Mozilla 的 Thunderbird。  ( 1 min )
